# Usage

To run the project, first clone the project

```
git clone https://github.com/the-refrigerator/robots-project-submission.git && cd robots-project-submission
```

Then install the dependencies

```bash
pip install -r requirements.txt
```

Start Jupyter Lab
```
jupyter lab
```

```
.
└── src
    ├── Derivation.ipynb # Derivation
    └── Simulation.ipynb # Simulation and visualization
```
